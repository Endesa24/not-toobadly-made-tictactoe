# TicTacToe

Bu tictactoe oyunudur
https://github.com/Endesa24/not-toobadly-made-tictactoe/blob/main/nottoobadlymadetictactoe.py

Kullanıcı "O" bilgisayar ise "X" olarak oynar.
Her zaman bilgisayar başlar ve bilgisayar rasgele oynar. Bilgisayarın ilk hamlesi bellidir ve hep ortaya oynar.
Sayılar ile oynanır. Dolu olan yerlere oynanamaz.
